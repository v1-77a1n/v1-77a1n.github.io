Website Evaluations

Frans Hal Museum

A website for the Frans Hal Museum that provides a little background on the exhibits available and a way to buy tickets.

  • Contrast: In terms of color contrast, the website's combination of colors made the elements on the website stand out. The different elements were used to section off important parts of the website;
  • Repetition: There's not much repetition; the colors don't repeat at all. There's some repetition with shapes; squares are used throughout the entire website.
  • Alignment: Every element on the website is aligned with each other, which helps a lot with the readability of the website.
  • Proximity: In terms of proximity, all related elements are grouped together/near each other. They're all separated by color, which overall helps with user interaction.

Arngren

A website selling various electronics, vehicles, and bicycles; site is seemingly Scandinavian in origin.

  • Contrast: In terms of contrast there's none whatsoever. The majority of the website is white and the little black that is used doesn't help with differentiating elements and sections from each other. There's also no spacing at all, which makes it an eyesore.
  • Repetition: There's no repeating patterns on the website at all. There are different shapes everywhere on the page; some images have borders, others don't, etc.
  • Alignment: A majority of the website's elements aren't aligned with each other. They're all pretty much all over the place; the items near each other aren't at the same alignment level.
  • Proximity: Some elements that are similar are grouped together, but that can't be said for everything that is similar on the website.
